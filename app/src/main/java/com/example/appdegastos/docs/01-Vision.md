# Pendify

## Objetivo

Pendify es una aplicación de finanzas personales diseñada para ayudar al usuario a comprender, organizar y mejorar su situación financiera.

No busca únicamente registrar ingresos y gastos.

Su objetivo es transformarse en un asistente financiero personal.

---

## Público objetivo

Personas que desean:

- controlar gastos
- administrar presupuestos
- ahorrar para objetivos
- conocer su salud financiera
- planificar gastos futuros
